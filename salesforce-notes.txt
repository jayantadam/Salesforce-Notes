lookup relationship - create a relationship to connects two objects loosely, allowing a child record to point to a parent record
 (like an Account to a Contact, or a custom object to a User) for data sharing, but the child can exist independently.
 it's defined on the child object by creating a new "Lookup Relationship" field, linking to the parent object, enabling easy data association without strict dependency or automatic deletion (unlike Master-Detail). 

Key Characteristics:
Loose Coupling: Child records aren't deleted if the parent is deleted (unless configured otherwise).
Optional: The lookup field on the child record doesn't have to be filled, allowing the child to exist without a parent.
One-to-Many: Typically links many child records to one parent record (e.g., many Contacts to one Account).
No Roll-Up Summaries: Cannot use Roll-Up Summary fields, which are a Master-Detail feature.
Creation: Created on the child object by selecting "Lookup Relationship" as the field type and choosing the parent object.

Types of OWD
          Organization-Wide Defaults (OWD) Types in Salesforce

          1.  Private  
              Users can only view and edit records they own or those shared with them. Most restrictive setting.

          2.  Public Read Only  
              All users can view all records, but only the owner or users with sharing can edit.

          3.  Public Read/Write  
              All users can view and edit all records. No ownership restrictions.

          4.  Public Read/Write/Transfer  
              Available for Leads and Cases. Users can view, edit, and transfer ownership of records.

          5.  Controlled by Parent  
              Used for child objects in a master-detail relationship. Access is inherited from the parent objectâ€™s OWD setting.

          6.  Public Full Access  
              Available for Campaigns. Users can view, edit, transfer, and delete all records.

          Special Cases  
          Price Books: Use, View Only, or No Access.  
          Knowledge Articles: Controlled by Data Category Visibility.
